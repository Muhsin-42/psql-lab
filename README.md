# Postgres SQL Editor

A simple in-browser SQL editor powered by PGlite.

## Features

- **PGlite**: Fully functional Postgres in the browser.
- **Modern UI**: Built with Next.js, Shadcn UI, and Tailwind CSS.
- **SQL Editor**: Write and execute SQL queries directly in your browser.

## Getting Started

First, install dependencies:

```bash
bun install
```

Then, run the development server:

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
